# Project-BeatSlash

Herní engine **KodEngine** a ukázkový projekt **BeatSlash**.

## Instalace

Projekt lze nainstalovat lokálně pomocí `pip`. V directoy projektu spusťte:

```bash
pip install .
```

To nainstaluje projekt a všechny potřebné dependence (`pygame`, `dearpygui`, `numpy`).

## Spuštění

Po instalaci můžete spustit editor příkazem:

```bash
kod-editor
```

Tím se spustí editor KodEngine s načteným projektem prozatím jen projekt BeatSlash.

Pokud příkaz není nalezen (není nastavený `PATH` na uživatelské scripts), lze editor spustit i takto:

```bash
python3 -m KodEngine.editor.Editor
```
