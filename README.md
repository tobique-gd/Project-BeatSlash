# Project-BeatSlash

Game engine **KodEngine** and showcase project **BeatSlash**.

## Instalation

You can install this project locally using `pip`. In the cloned directory run:

```bash
pip install .
```

That will install the project and all needed dependencies (`pygame`, `dearpygui`, `numpy`).

## Running

After instalation you can run the editor using:

```bash
python3 -m KodEngine.editor.Editor
```
