# DynamicBody2D

DynamicBody2D represents a physics body with integrated velocity.

| Function | Description |
| --- | --- |
| __init__() | Initializes body state and velocity field. |

| Property | Description |
| --- | --- |
| position | Local physics body position inherited from Node2D. |
| rotation | Local rotation inherited from Node2D. |
| z_index | Render ordering inherited from Node2D. |
| velocity | Linear velocity applied by the physics server. |
